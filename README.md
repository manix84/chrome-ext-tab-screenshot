# chrome-ext-tab-screenshot

A simple Chrome extension that takes a screenshot of the current tab and saves it directly to your default download directory — no fuss, no frills.

## Overview

Click the extension icon in your browser toolbar and a screenshot of the active tab is instantly captured and saved to your default downloads folder. That's it.

## Motivation

Many screenshot extensions on the Chrome Web Store require you to:

- Create an account before you can use them
- Accept invasive data collection or tracking (spyware)
- Wade through unnecessary options and bloated UIs

This extension was built to cut through all of that. It does exactly one thing, does it well, and nothing else.

## How to Use

1. Install the extension in Chrome.
2. Navigate to the page you want to capture.
3. Click the extension icon in the toolbar.
4. The screenshot is automatically saved to your default download directory.
