# Privacy Policy

**Effective Date:** 26 March 2026

This extension does not collect, store, transmit, or share any personal data.

The extension performs a single function: capturing a screenshot of the currently visible browser tab when the user explicitly clicks the extension icon.

## How it works

- The screenshot is captured only after a direct user action (clicking the extension icon).
- The image is processed locally within the browser.
- The screenshot is saved directly to the user’s device using the browser’s download functionality.
- No data is sent to any external servers.

## Data usage

- No personal data is collected or stored.
- No browsing history is tracked.
- No user activity is monitored.
- No analytics or tracking technologies are used.
- No third-party services are involved.

## Data sharing

This extension does not sell, share, or transfer any user data to third parties.
